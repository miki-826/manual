# UI Visual Brief

## Visual Concept
紙の操作メモと赤ペン校正がそのまま画面になった、軽い業務ツール。

## Materials
- 背景: 淡い方眼紙、紙繊維、低コントラストの作業面。
- Control: 赤い校正ラベル風の主操作ボタン外装。文字は焼き込まずHTMLで載せる。
- Motif: 赤ペンの丸、矢印、チェック。初期画面の小さな印象づけに使う。

## Color Roles
- Base: `#f7f7f4`
- Surface: `#ffffff`
- Text: `#252525`
- Secondary text: `#5f5a52`
- Accent: `#d92727`

## Layout
最初は大きな貼り付け領域だけを見せる。編集後は中央に画像、右側にコピー/共有/説明文/Undoの順で配置する。390px幅では縦積みで操作が自然に下へ流れる。

## Avoid
紫青グラデーション、ガラスカード、発光、AIチャットUI、装飾過多、画像に文字を焼き込むこと。
